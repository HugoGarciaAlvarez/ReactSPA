# Guía del proyecto

## Cómo ejecutar el proyecto

Primero clona el proyecto

```bash
git clone [https://github.com/HugoGarciaAlvarez/ReactSPA.git](https://github.com/HugoGarciaAlvarez/ReactSPA.git)
cd react-app
```
A continuación descarga las dependencias de node_modules
```bash
npm install
```
Lanza el servidor de desarrollo
```bash
npm run dev
